(static, constant) MEDIA_BEFORECOMPLETE

Fired just before the player completes playing. At this point the player will not have moved on to either showing the replay screen or advancing to the next playlistItem. This is the ideal moment to insert postroll ads using the playAd() API

Fired just before the player begins playing. At this point the player will not have begun playing or buffering. This is the ideal moment to insert preroll ads using the playAd() API
